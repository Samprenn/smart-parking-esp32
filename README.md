# 🚗 Sistem Parkir Cerdas Berbasis IoT

## 📌 Deskripsi
Project ini merupakan sistem parkir otomatis berbasis ESP32 dan sensor ultrasonik untuk mendeteksi kendaraan masuk dan keluar secara real-time.

## ⚙️ Komponen
- ESP32
- HC-SR04
- Servo SG90
- LCD I2C
- WiFi Monitoring

## 🚀 Fitur
- Monitoring slot parkir
- Portal otomatis
- Cloud monitoring
- Notifikasi parkir penuh
